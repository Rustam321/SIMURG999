## Соглашения и приёмы

- **Иконки** — CSS-маски: `-webkit-mask-image`/`mask-image` с URL-encoded SVG data URI, `background: var(--accent-blue)`; замена присланного SVG через python (re.subn по классу `.module-item__icon--name::before` и т.п.). Filled-SVG → `fill='%23000'`; stroke-SVG → `fill='none' stroke='%23000'`.
- **Цветной/декоративный SVG стилизовать**: выкинуть заливки и декор-точки, оставить смысловые формы, сделать монохром.
- **Замена фото/эмблемы пользователем**: файлы приходят картинками В ЧАТ, файлами не сохраняются — просить положить в `img/` с конкретным именем; я вижу их, но вставить могу только реальный файл на диске.
- **Дизайн-токены**: --accent-blue #3da9fc, --accent-cyan #4fd1e8, --accent-green #3ddc97, --accent-purple #a78bfa, --accent-orange #f5a623, --bg-base #070b14, --bg-deep #050810; фамилия в карточках — золото #e0b45a
- **Шрифты**: Manrope (текст) + Rajdhani (заголовки)
- **Скролл-анимации**: IntersectionObserver, `.reveal`/`.is-visible`, каскад по индексу среди siblings (90ms). Новый класс-цель добавлять в селектор в js/main.js!
- **Рендер PDF-макетов**: `pip install --user pymupdf` (fitz), `doc[i].get_pixmap(matrix=Matrix(s,s)).save(...)` — poppler/Quartz недоступны.
- **Обрезка белых полей у фонов**: PIL, скан яркости строк/столбцов (порог ~100–110), crop.
- **Блокировка файлов (было 2026-07-12)**: `index.html`/`style.css` иногда залипают на `Operation not permitted` (чтение и запись) — вероятно TCC/iCloud-синк папки Documents. Каталог при этом на запись доступен (новые файлы создаются) → обход: писать во временный файл + `mv`.
